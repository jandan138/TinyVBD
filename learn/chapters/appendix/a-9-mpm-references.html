MPM 参考文献

「MPM 篇」（Part 13）是全书第一个真正跳出 mesh 的范式：material point 携带物理状态、永久 Lagrangian；background grid 只是每步借来算力、用完即弃的 scratch。这一页把它的源头分三层列清楚——经典奠基（Sulsky、Stomakhin 雪、APIC、MLS-MPM、那门标准 course）、以及前沿的三条现代路线（MPM Lite / PB-MPM / CK-MPM）。前沿部分本教程只描述机制、给出来源，不替它们背书任何精确公式、加速比或网格 / 粒子规模——要数字请自己点进项目页与 PDF。

经典 Classic

  • Application of a Particle-in-Cell Method to Solid Mechanics（原始 MPM）. Deborah Sulsky, Zhen Chen, Howard L. Schreyer（1994）。 MPM 的奠基之作：把流体里的 PIC / FLIP 思路搬到固体力学，确立「物质点载状态 + 背景网格借力」的混合 Lagrangian-Eulerian 骨架。想知道 13-2 的 P2G / grid solve / G2P / advect 这套循环从何而来，回到这里。 DOI：10.1016/0045-7825(94)90112-0。
  • A Material Point Method for Snow Simulation. Alexey Stomakhin, Craig Schroeder, Lawrence Chai, Joseph Teran, Andrew Selle. ACM TOG（SIGGRAPH 2013）。 把 MPM 带进图形学主流的那一篇（也是《冰雪奇缘》雪效背后的方法）：用弹塑性本构 + MPM 捕捉雪的压实、断裂、堆积。想看 MPM 为什么天生擅长「mesh 会撕裂 / 缠死」的材料，从它的视频和论文进。 项目页：math.ucla.edu/~jteran · SSCTS13 · DOI：10.1145/2461912.2461948。
  • The Affine Particle-In-Cell Method（APIC）. Chenfanfu Jiang, Craig Schroeder, Andrew Selle, Joseph Teran, Alexey Stomakhin. ACM TOG（SIGGRAPH 2015）。 13-3 的主角：PIC 太耗散（丢角动量、糊）、FLIP 太吵；APIC 在粒子上多带一个仿射速度场 $C_p$，于是传输既保角动量又低耗散——成为现代 MPM 的默认传输。想弄清「传输是 MPM 的灵魂与软肋」，读它。 DOI：10.1145/2766996。
  • A Moving Least Squares Material Point Method with Displacement Discontinuity and Two-Way Rigid Body Coupling（MLS-MPM）. Yuanming Hu, Yu Fang, Ziheng Ge, Ziyin Qu, Yixin Zhu, Andre Pradhana, Chenfanfu Jiang. ACM TOG（SIGGRAPH 2018）。 用 moving least squares 把 APIC 的传输与内力计算统一进同一套数学，实现更简洁、更快，是当下最常被拿来当 baseline / 教学起点的现代 MPM（Taichi 的 88 行 MLS-MPM 即出于此）。13-3 / 13-4 的实现口径多半参照它。 DOI：10.1145/3197517.3201293 · 作者页：yuanming.taichi.graphics。
  • The Material Point Method for Simulating Continuum Materials. Chenfanfu Jiang, Craig Schroeder, Joseph Teran, Alexey Stomakhin, Andrew Selle. SIGGRAPH 2016 course。 标准入门 course：把 P2G / G2P、核（B-spline）、$F$ 更新、本构、显式与隐式求解系统讲一遍。读 13-2~13-5 之前先翻它建立全局，能省下大量「这些步骤为什么这样排」的疑惑。 course PDF：math.ucla.edu/~cffjiang · mpmcourse.pdf。

前沿 Frontier

下面三条是 MPM 的现代加速 / 重写路线，分别换了设定、解决不同问题。本教程对它们只描述机制、给来源——任何精确公式、加速倍数、网格 / 粒子规模都没有独立核对，13-6~13-8 正文里也是这个口径。要可引用的数据，请回到各自的项目页 / PDF 自行核对。

  • MPM Lite（Linear Kernels and Integration without Particles）. 项目页 mpmlite.github.io。 13-6 的主讲对象。机制（仅描述、不背书数字）：让粒子主要作运动状态（kinematic state）的载体，求解时去掉基于粒子的求积（quadrature）、改用线性核，从而砍掉「标准隐式 MPM 开销 ∝ 每格粒子数 PPC × 宽核 stencil」这个瓶颈，同时保留 MPM 对多材料的通用性。具体加速比 / 规模 / 作者列表请以项目页为准。 项目页：mpmlite.github.io。
  • PB-MPM（Position-Based Material Point Method）. EA SEED，SIGGRAPH 2024，Chris Lewin。 13-7 的对象，也是本篇接回 Part 2 的 PBD / XPBD 的地方：把 MPM 重写成 position-based / 约束求解问题，像 PBD 那样允许更大步长、更稳，面向游戏物理实时性。同一个「掰位置满足约束」的哲学，从 2-1 / 2-2 搬到了 material point 上。机制可讲，精确数值以官方 PDF 为准。 EA SEED 页：ea.com/seed。
  • CK-MPM（A Compact-Kernel Material Point Method）. arXiv:2412.10399。 13-8 的对象。机制（仅描述）：构造一个 $C^2$ 连续的紧支撑（compact）核，在稳定性 / 精度 / 效率间取新平衡；可与 APIC / MLS-MPM 无缝衔接，每粒子关联的网格节点数约为线性核的 2×（远少于二次 B-spline 的宽 stencil）；可用 Taichi 或开源 GPU MPM 框架实现。精确核形式与数值以论文为准。 arXiv：arXiv:2412.10399。

怎么配着书读 How to Read

  • 先 2016 course + Sulsky 1994 把循环（P2G / grid / G2P / advect）和起源接上，对照 13-1（为什么 MPM）/ 13-2（一步循环）。
  • 再 APIC 2015 + MLS-MPM 2018 吃透传输，对照 13-3；本构 $\psi(F_p)\to P_p$ 复用 FEM 篇（9-3 / 9-4），见 13-4。
  • 隐式 MPM（13-5）是网格 DOF 上的又一个 $\min G(x)$，和 1-2、VBD（Part 3）、IPC 全局 Newton 同源；前沿三篇按上节的 DO-NOT-STATE 纪律读机制、不背数字。
关于前沿数字的诚实声明

经典 MPM（P2G / G2P、APIC、$F$ 更新、隐式求解）是成熟内容，本页按定论引用，可信。但前沿三篇（MPM Lite / PB-MPM / CK-MPM）的精确公式、加速倍数、网格与粒子规模、确切作者列表，本教程没有独立核对——上面只描述了机制并给出项目页 / arXiv，正文 13-6~13-8 也守同一纪律。需要可引用的数据，请回到各自的项目页与 PDF 自行核对。13-6 的手算是「为讲解构造的自洽示例、非论文复现」。另外，经典几篇的确切卷期 / 年份若与你查到的略有出入，以 DOI / 项目页为准。
