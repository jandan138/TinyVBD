术语表 Glossary

这本书刻意保留了所有英文术语，方便你无缝衔接论文和代码。这里把它们集中起来，每个配一句中文解释和「在哪一节第一次认真讲到」的链接，供你随时回查。

核心方法与最小化框架

术语中文解释详见
Vertex Block Descent (VBD) 本书主角。把隐式欧拉的最小化问题，用「一次只动一个顶点的 3 自由度」的 block coordinate descent 来解；primal、保留完整局部 Hessian、$G$ 单调下降。 0.1 · 3.1
incremental potential / variational implicit Euler 增量势能 / 隐式欧拉的变分真身。一步隐式欧拉等价于最小化 $G(\mathbf x)=\frac{1}{2h^2}\lVert\mathbf x-\mathbf y\rVert_M^2+E(\mathbf x)$。全书地基。 1.2
block coordinate descent 块坐标下降。把变量分成块（这里一块 = 一个顶点的 3 个坐标），轮流固定其余、只优化当前块。VBD 的精髓。 3.1
primal vs dual primal（原始）求解器直接在位置变量上下降（VBD）；dual（对偶）求解器解约束力 / 拉格朗日乘子（XPBD）。高质量比下二者命运迥异。 5.1

物理与数学量

术语中文解释详见
inertia / predicted position $\mathbf y$ 惯性项 / 预测位置。$\mathbf y=\mathbf x^t+h\mathbf v^t+h^2\mathbf a_{ext}$，即「纯靠惯性加外力会漂到哪」。惯性项 $\frac{1}{2h^2}\lVert\mathbf x-\mathbf y\rVert_M^2$ 是 $G$ 的一半。 1.4
Hessian 能量对位置的二阶导矩阵。顶点 Hessian $\mathbf H_i=\frac{m_i}{h^2}\mathbf I+\sum_j\partial^2 E_j/\partial\mathbf x_i^2$，VBD 牛顿步要解的 3×3 矩阵。 1.5 · 3.3
SPD / positive definite 对称正定。Hessian 正定时局部能量是个有唯一谷底的碗。惯性项恒正定，是 VBD 不必强制 SPD 投影也能稳的关键。 3.6
Newton's method 牛顿法。用 Hessian 解 $\Delta\mathbf x=\mathbf H^{-1}\mathbf f$。VBD 对每个顶点做一步局部牛顿；全局 Newton 则解整个大稀疏系统。 3.3
line search / Armijo condition 线搜索 / Armijo 充分下降。Newton 先给方向，再用回溯步长 $\alpha$ 试探着走，直到能量下降量足够；IPC 的 CCD line search 是它的接触感知升级版。 3.4 · 12.5
stiffness $k$ / rest length $l_0$ 弹簧刚度 / 自然长度。弹簧能量 $E=\frac12 k(l-l_0)^2$，$l=\lVert\mathbf x_a-\mathbf x_b\rVert$。 1.5
high mass ratio 高质量比。链上轻重质点相差很多倍（TinyVBD 默认末端重 1000×）。XPBD 会崩，VBD 不崩。 5.1
high stiffness ratio 高刚度比。相邻弹簧软硬相差极大（如 1:10000）。这是 VBD 这种局部方法的软肋，收敛慢。 5.2
substep 子步。把一帧的 $h$ 再切成若干份分别积分，等价于另一种「加迭代」的方式，常比加迭代更省。 4.5 · 5.3
skip spring (bending) 跳跃弹簧。连接顶点 $i$ 与 $i+2$ 的额外弹簧，用来近似弯曲刚度。TinyVBD 用 stiffnessSkipSpring 控制。 5.3

求解顺序与家族成员

术语中文解释详见
Gauss-Seidel 逐个更新、立刻用上刚算出的新值的扫描顺序。收敛快，但有顺序依赖。VBD 的 sweep 用它。 3.5
Jacobi 所有点用同一份旧值并行更新。易并行但收敛慢。VBD 靠图着色逼近 Jacobi 的并行度而保持 Gauss-Seidel 的收敛。 3.5
graph coloring 图着色。给顶点染色使相邻点不同色；同色顶点可并行更新。VBD 着色的是顶点图，颜色数远少于 XPBD 的约束图。 3.5
PBD (Position Based Dynamics) 预测位置后把粒子顺序投影到约束上、反复迭代。病：有效刚度依赖迭代次数与时间步，不收敛到良定义解。 2.1
XPBD 扩展 PBD。引入 compliance $\alpha$ 与拉格朗日乘子 $\lambda$，让刚度变成时间步 / 迭代无关的物理量。本质是隐式欧拉 KKT 系统的 Gauss-Seidel。 2.2
compliance $\alpha$ / Lagrange multiplier $\lambda$ 柔度 $\alpha=1/k$（刚度的倒数）；拉格朗日乘子 $\lambda$ 是约束力的对偶变量。XPBD 用 $\tilde\alpha=\alpha/h^2$ 累积 $\lambda$。 2.2
Projective Dynamics (local/global) 把能量写成特殊二次型，local 步并行投影、global 步解一个矩阵恒定、可预分解（Cholesky）的线性系统。快，但只能表达这种特殊能量、线性收敛。 2.3

加速与谱半径

术语中文解释详见
Chebyshev semi-iterative acceleration 切比雪夫半迭代加速（Wang 2015）。在每次完整 sweep 后做一次外推，用一组 $\omega_n$ 递推系数把收敛提速。TinyVBD 的 applyAccelerator。 4.4
spectral radius $\rho$ 谱半径。迭代矩阵最大特征值的模的估计值，$\rho\in(0,1)$。Chebyshev 系数靠它算；$\rho$ 太大会 overshoot / 振荡甚至发散。 4.4

接触与 IPC（Part 12）

原书那根 strand 没有碰撞，所以上面几张表里没有任何接触术语。Part 12「IPC 篇」一口气引入了一整套接触词汇——下面这些就是你读到 log-barrier / CCD / PSD 投影时会想回查的条目。IPC 一侧的论文出处见 a-7 参考文献。

术语中文解释详见
Incremental Potential Contact (IPC) 增量势能接触：把接触写成 primal interior-point 的 log-barrier、注入 $\min G(\mathbf x)$，用 CCD-filtered line search 保证每步 intersection-free。 12.1 · 12.2
interior-point method 内点法：在可行域内部迭代、靠 barrier 阻止越界，永远不碰约束边界。IPC 处理接触的范式。 12.1
log-barrier $b(d,\hat d)$ 对数障碍能量：作用于无符号距离 $d$，$d\to0$ 时发散到 $\infty$、$d\ge\hat d$ 时为 0，设计成 $C^2$ 以进 Newton。 12.2
activation distance $\hat d$ 激活阈值：距离 $<\hat d$ 才开始发力（barrier 非零）的用户参数。 12.2
inversion-free 不翻转保证：单个 FEM 单元保持正取向，$J=\det\mathbf F>0$。它是材料单元内部的保证，和 surface primitive 之间的 penetration-free 不同。 9.2 · 10.3
intersection-free 无穿透保证：IPC 的核心卖点——不是事后兜底，而是整个非线性迭代中始终保持可行。 12.1 · 12.5
unsigned distance $d$ 无符号距离：一对 surface primitive（点-三角 / 边-边）间的最近距离，barrier 的自变量。 12.3
distance type / point-triangle / edge-edge 距离类型：接触按几何分点-三角与边-边两大类；最近点可退化到 point-edge / point-point。 12.3
barycentric coordinates $\lambda_a,\lambda_b,\lambda_c$ 重心坐标：最近点在三角上的权重；距离梯度按它落到各顶点（「谁近分得多」）。 12.3
PSD projection 半正定投影：把局部 Hessian 的负特征值钳到 0（$\max(0,\lambda_i)$）保证下降方向；接回原书 10-2 的 SPD 投影。 12.4
CCD (continuous collision detection) 连续碰撞检测：沿运动轨迹求最大不穿透步长 $t^\star$，是 intersection-free 的几何来源。 12.5
filtered line search CCD 过滤线搜索：用 $t^\star$ 把 Newton 步长截进可行域，$\alpha=\min(\alpha_N, c\,t^\star, \alpha_{CFL})$。 12.5
CFL bound $\alpha_{CFL}$ CFL 式保守接触界：进一步限制单步位移的保守上限。 12.5
Coulomb friction (lagged / mollified) 库仑摩擦：IPC 在同一优化框架内处理，用 lagged potential 固定法向力/接触集、并把静-动过渡光滑化（mollified）成可微。 12.6
mollifier 软化子：随退化程度（如 edge-edge 平行）衰减的权重，压住病态梯度贡献。 12.3 · 12.6
global sparse Newton 全局稀疏牛顿：IPC 每次迭代解整个稀疏系统 $\mathbf H\mathbf p=\mathbf f$，对照 VBD 的 per-vertex 3×3 局部牛顿。 12.8 · 12.9
CHOLMOD 稀疏 Cholesky 库（挂 MKL LAPACK/BLAS），IPC 做符号分解 + 并行数值分解用它。 12.9
CSR 压缩稀疏行格式：装配全局 Hessian 的存储格式。 12.9
