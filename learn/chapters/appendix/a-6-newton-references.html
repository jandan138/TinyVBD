Newton / VBD / AVBD 参考文献

「Newton 篇」把读者从一根 strand 带到 GPU 上一整块 cloth，再带到 AVBD 的硬约束世界。这一页把这条路上真正的源头列出来：两篇论文、它们背后的引擎与框架、以及能让你「看见」算法在跑的 demo 与视频。每条都附一句中文，告诉你它值不值得点进去、该带着什么问题去读。

论文 Papers

  • Vertex Block Descent. Anka He Chen, Ziheng Liu, Yin Yang, Cem Yuksel. ACM TOG 43(4), Art. 116（July 2024）。 本书与 Newton particle 求解器共同的主角：把隐式欧拉的变分形式做成顶点级的 block coordinate descent，每顶点一个 3×3 牛顿步、按顶点图着色做 Gauss-Seidel，unconditionally stable。想弄清 Newton 的 particle 路径「为什么这样写」，先读它。 arXiv：arXiv:2403.06321 · DOI：10.1145/3658179 · 项目主页：graphics.cs.utah.edu/research/projects/vbd（作者镜像页：ankachan.github.io/Projects/VertexBlockDescent）。
  • Augmented Vertex Block Descent (AVBD). Chris Giles, Elie Diaz, Cem Yuksel. ACM TOG 44(4), Art. 90（Aug 2025）。 在 VBD 骨架上叠加 augmented Lagrangian 与自适应 primal-dual，从而能强制近无穷刚度的硬约束、并在高刚度比下收敛得更好——正是 Newton 给 rigid 部分选的解法。想理解 penalty ramping（warm-start、衰减、封顶、增长）这套机制，读它。 DOI：10.1145/3731195 · 项目主页：graphics.cs.utah.edu/research/projects/avbd。
  • A Chebyshev Semi-Iterative Approach for Accelerating Projective and Position-Based Dynamics. Huamin Wang. ACM TOG（SIGGRAPH Asia 2015）。 TinyVBD 原书的 Chebyshev 加速出自这里——但请注意一个对照点：Newton 的 particle 路径并没有用它，只跑按 color 的 Gauss-Seidel。放在这里是为了把「书里学过的加速」和「Newton 实际代码」之间的差异说清楚（见 4.4）。

引擎与框架 Engine & Framework

  • Newton. GPU 加速物理引擎，建在 NVIDIA Warp 之上，面向 robotics / 仿真研究；其 SolverVBD 对 particle 用 VBD、对 rigid 用 AVBD，是「Newton 篇」逐行走读的对象。先看文档主页建立全局，再进仓库读 solver_vbd.py 与 kernels。 文档：newton-physics.github.io/newton · 仓库：github.com/newton-physics/newton · NVIDIA 开发者页：developer.nvidia.com/newton-physics。
  • NVIDIA Warp. Newton 的 GPU kernel 语言 / Python 框架：@wp.kernel + wp.launch(dim=N) 的 SPMD 模型、wp.array/wp.struct、以及 CUDA graph capture。不懂 Warp 就读不懂 Newton 的并行 sweep，建议在读 7.2 前后翻一遍它的文档。 开发者页：developer.nvidia.com/warp-python · 仓库：github.com/NVIDIA/warp。
  • OpenUSD. Newton 的场景 / 资产描述基座，也是你从 Isaac 背景过来最熟的一块。理解 Model（静态资产）与 State（动态属性）的分离，从 USD 的视角切入会很顺。 主页：openusd.org。

demo 与视频 Demos & Video

  • AVBD 官方视频. 论文配套的可视化，硬约束、堆叠、摩擦、关节都在动里一目了然——读 AVBD 论文前先看它，能省下很多想象成本。 视频：youtube.com/watch?v=bwJgifqvd5M。
  • AVBD 2D 参考实现. Chris Giles。一份可读的 2D reference demo，把 primal-dual 与 penalty 演化压到最小、最透明的形态——想动手跟着 step through，这份比工业引擎好读得多（呼应 11.1 的 widget）。 仓库：github.com/savant117/avbd-demo2d。
  • Cem Yuksel 研究主页. VBD 与 AVBD 的共同作者之一；他的研究索引页是顺藤摸瓜找最新论文、视频与代码的总入口。 主页：cemyuksel.com/research。
