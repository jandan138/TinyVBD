solve()：一次 Gauss-Seidel sweep 逐行解剖

这是全书所有铺垫的兑现点。solve() 一次调用 = 一遍 Gauss-Seidel sweep：从头到尾扫过每个顶点，给每个顶点解一个 3×3 牛顿步。整本书讲的 $G$、$\mathbf f_i$、$\mathbf H_i$、弹簧 Hessian，全在这二十几行里。我们一行一行把它和公式对上。

先看全貌

把 solve() 完整放出来，然后拆：

main.cpp · solve()
void solve() {
    float dtSqrReciprocal = 1.f / (params.dt * params.dt);
    auto& edges = strand.edges;
    for (size_t iV = 1; iV < strand.numVerts; iV++)
    {
        Mat3 h = Mat3::Zero();
        Vec3 f = Vec3::Zero();

        f = strand.vertexMass(iV) * (strand.inertia.col(iV) - strand.vertex(iV))* (dtSqrReciprocal);
        h += strand.vertexMass(iV) * dtSqrReciprocal * Mat3::Identity();
        for (size_t iNeiE = 0; iNeiE < strand.vertAdjacentEdges[iV].size(); iNeiE++)
        {
            int edgeId1 = strand.vertAdjacentEdges[iV][iNeiE];

            if (edgeId1 != -1)
            {
                IdType v1 = edges[edgeId1][0];
                IdType v2 = edges[edgeId1][1];
                Vec3 diff = strand.mVertPos.col(v1) - strand.mVertPos.col(v2);
                FloatingType l = diff.norm();
                FloatingType l0 = strand.orgLengths(edgeId1);
                // evaluate hessian
                Mat3 h_1_1;
                FloatingType stiffness_ = strand.edgesStiffness[edgeId1];
                h_1_1 = stiffness_ * (Mat3::Identity() - (l0 / l) * (Mat3::Identity() - diff * diff.transpose() / (l * l)));
                h += h_1_1;

                if (v1 == iV)
                {
                    f += (stiffness_ * (l0 - l) / l) * diff;

                }
                else {
                    f -= (stiffness_ * (l0 - l) / l)* diff;
                }

            }
        }

        Vec3 dx = h.colPivHouseholderQr().solve(f);

        strand.vertex(iV) += dx;

    }
}

骨架是：外层循环扫每个顶点 → 为它备一个 3×3 的 h 和 3×1 的 f → 先填惯性项 → 内层循环把每条邻接弹簧的贡献累加进去 → 解 3×3 线性系统得到 dx → 就地挪动顶点。这正是 3.3 的 $\Delta\mathbf x_i = \mathbf H_i^{-1}\mathbf f_i$。下面逐块对公式。

外层循环与那个 1/dt²

solve() · 顶层
float dtSqrReciprocal = 1.f / (params.dt * params.dt);
for (size_t iV = 1; iV < strand.numVerts; iV++)

dtSqrReciprocal 就是 $1/h^2$，全函数复用。循环从 iV=1 起——跳过 0 号 pinned 顶点（它被 4.2 钉死，不参与求解）。每个 iV 是一次独立的「block」：只动这一个顶点的 3 个自由度，其余全当固定。这就是 block coordinate descent（3.1）落到代码里的样子。

惯性项：f 和 h 的第一块

solve() · 惯性项
f = strand.vertexMass(iV) * (strand.inertia.col(iV) - strand.vertex(iV))* (dtSqrReciprocal);
h += strand.vertexMass(iV) * dtSqrReciprocal * Mat3::Identity();

这两行就是 3.3 里 $\mathbf f_i$、$\mathbf H_i$ 的惯性部分：

$$\mathbf f_i \mathrel{+}= \frac{m_i}{h^2}\big(\mathbf y_i - \mathbf x_i\big),\qquad \mathbf H_i \mathrel{+}= \frac{m_i}{h^2}\mathbf I$$

对照：vertexMass(iV) 是 $m_i$，inertia.col(iV) 是惯性目标 $\mathbf y_i$，vertex(iV) 是当前位置 $\mathbf x_i$，dtSqrReciprocal 是 $1/h^2$。力 $\mathbf f$ 是负梯度：$-\nabla_i G$ 的惯性部分是 $-\frac{m_i}{h^2}(\mathbf x_i-\mathbf y_i)=\frac{m_i}{h^2}(\mathbf y_i-\mathbf x_i)$，正是代码里 (inertia - x) 的方向——它把顶点往惯性碗底 $\mathbf y_i$ 拉。Hessian 的惯性部分 $\frac{m_i}{h^2}\mathbf I$ 恒正定，是 3.6 里「即使弹簧 Hessian 不定也能稳」的那块定海神针。

内层循环：每条邻接弹簧的贡献

现在遍历这个顶点的邻接弹簧——用的正是 4.1 里的 vertAdjacentEdges[iV]（force elements 集合 $\mathcal F_i$）。先取边、取几何量：

solve() · 取边
int edgeId1 = strand.vertAdjacentEdges[iV][iNeiE];
if (edgeId1 != -1)
{
    IdType v1 = edges[edgeId1][0];
    IdType v2 = edges[edgeId1][1];
    Vec3 diff = strand.mVertPos.col(v1) - strand.mVertPos.col(v2);
    FloatingType l = diff.norm();
    FloatingType l0 = strand.orgLengths(edgeId1);
    FloatingType stiffness_ = strand.edgesStiffness[edgeId1];

v1, v2 是这条弹簧的两端，diff $=\mathbf d=\mathbf x_{v1}-\mathbf x_{v2}$ 是弹簧向量，l $=\lVert\mathbf d\rVert$ 是当前长度，l0 $=l_0$ 是 rest 长度，stiffness_ $=k$ 是这条边的刚度。和 1.5 的记号一一对应。（edgeId1 != -1 是个安全判：邻接表里 -1 代表空槽，本 strand 的 from() 不会塞 -1，但留着保险。）

弹簧 Hessian：h_1_1

solve() · 弹簧 Hessian
h_1_1 = stiffness_ * (Mat3::Identity() - (l0 / l) * (Mat3::Identity() - diff * diff.transpose() / (l * l)));
h += h_1_1;

这一行是 1.5 推出来的弹簧 Hessian，直接抄进代码：

$$\mathbf H_{aa} = k\Big[\mathbf I - \frac{l_0}{l}\Big(\mathbf I - \frac{\mathbf d\mathbf d^\top}{l^2}\Big)\Big]$$

对照：diff * diff.transpose() 是外积 $\mathbf d\mathbf d^\top$，除以 (l*l) 即 $/l^2$。整块乘 stiffness_=$k$。把它的物理读出来：沿弹簧方向的特征值是 $k$（拉伸方向永远硬）；垂直方向的特征值是 $k(1-l_0/l)$——当弹簧被压缩（$l<l_0$）时这个值为负，$\mathbf H_{aa}$ 不定。这正是 3.6 里那个「弹簧单独看可能不定」的来源。VBD 不对它做 SPD 投影，靠的就是上面那块恒正定的惯性 Hessian 把整个 h 兜回正定附近。注意代码无脑 += 累加、不做任何 clamp——这是 VBD 的故意设计。

弹簧力：那个 ± 号

solve() · 弹簧力
if (v1 == iV)
{
    f += (stiffness_ * (l0 - l) / l) * diff;
}
else {
    f -= (stiffness_ * (l0 - l) / l)* diff;
}

弹簧对端点 $a$ 的力是 1.5 的

$$\mathbf f = k\,\frac{l_0-l}{l}\,\mathbf d,\qquad \mathbf d=\mathbf x_{v1}-\mathbf x_{v2}$$

系数 stiffness_*(l0-l)/l 就是 $k\frac{l_0-l}{l}$，乘上 diff=$\mathbf d$。± 号取决于当前顶点 iV 是这条弹簧的哪一端：diff 是按 $\mathbf x_{v1}-\mathbf x_{v2}$ 定义的，所以它天然是「对 v1 的力」。若 iV == v1，直接 +=；若 iV 是另一端 v2，力反向，-=。物理含义很直白：弹簧被拉长（$l>l_0$，$l_0-l<0$）时，力把两端往中间拽；被压短时往外推。每条邻接弹簧都把它对 iV 的那一份力加进 f，加完就是完整的 $\mathbf f_i$。

为什么 H 只累加 h_1_1、不管 ±号

力是向量、有方向，换端点要翻号；但 Hessian $\mathbf H_{aa}$ 对两个端点是同一个 3×3 块（$\partial^2 E/\partial\mathbf x_a^2 = \partial^2 E/\partial\mathbf x_b^2$，因为弹簧能量只依赖 $\mathbf d=\mathbf x_a-\mathbf x_b$）。所以 h += h_1_1 不分 v1/v2，无条件累加——这不是偷懒，是对的。

3×3 牛顿步与就地更新

solve() · 解 & 更新
Vec3 dx = h.colPivHouseholderQr().solve(f);
strand.vertex(iV) += dx;

累加完，h 是完整的 $\mathbf H_i$、f 是完整的 $\mathbf f_i$。最后两行就是一步牛顿：

$$\Delta\mathbf x_i = \mathbf H_i^{-1}\mathbf f_i,\qquad \mathbf x_i \mathrel{+}= \Delta\mathbf x_i$$

colPivHouseholderQr().solve(f) 是 Eigen 解这个 3×3 系统 $\mathbf H_i\,\Delta\mathbf x = \mathbf f_i$ 的方式（列主元 QR，对 3×3 又稳又快，不必显式求逆）。strand.vertex(iV) += dx 借助 4.1 的「返回列引用」就地把顶点挪过去。就地是关键：循环里下一个顶点 iV+1 计算它的 diff 时，立刻读到 iV 的新位置——这就是 Gauss-Seidel（用最新值），而非 Jacobi（统一用旧值）。3.5 讲的就是怎么靠图着色在 GPU 上既保 Gauss-Seidel 的收敛、又拿到并行。

把这一步 3×3 放到全程尺度（呼应 0-2）

一次 sweep = numVerts-1 次这样的 3×3 QR 解（跳过 pinned 0 号）。默认 20 顶点 strand ⇒ 每 sweep 19 次；每帧 100 次迭代、共 299 帧解算，整次运行约 $19\times100\times299\approx 5.7\times10^5$ 次 3×3 求解。每次都廉价到近乎免费——这正是 block descent 用「小而多」换掉 Newton「大而少」的算术（数字为默认参数下的自洽示例）。

一次 sweep 还不够：solve() 要被调很多次

solve() 只走一遍所有顶点（一次 sweep）。4.5 的主循环每个 substep 会调它 numIterations 次（默认 100），让 $G$ 一遍遍下降、逼近 $\arg\min G$。单次 sweep 不解出精确解，但 3.2 保证了每个局部牛顿步都让全局 $G$ 单调下降——所以反复扫就稳稳收敛。想看慢动作的逐顶点能量下降，回 3.1 的 vbd-sweep 部件。

这一节你要带走的

把整段映射一句话记牢：外层扫顶点（跳过 pinned 0）→ 惯性项给 $\mathbf f_i,\mathbf H_i$ 打底（$\frac{m}{h^2}(\mathbf y-\mathbf x)$、$\frac{m}{h^2}\mathbf I$，恒正定）→ 遍历 vertAdjacentEdges，每条弹簧累加 Hessian h_1_1（= $k[\mathbf I-\frac{l_0}{l}(\mathbf I-\frac{\mathbf d\mathbf d^\top}{l^2})]$）和带 ± 号的力（= $k\frac{l_0-l}{l}\mathbf d$）→ dx = H⁻¹f 解 3×3 → vertex(iV) += dx 就地写回（Gauss-Seidel）。前面 1.2/1.4/3.3 推的每个公式，在这里都能用手指点到对应的那一行。
