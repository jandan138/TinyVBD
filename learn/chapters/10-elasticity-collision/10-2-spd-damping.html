SPD 投影与 Rayleigh damping

在 TinyVBD 那根 strand 上，我们做了一个大胆的决定：解 $\Delta\mathbf x_i=\mathbf H_i^{-1}\mathbf f_i$ 时不对局部 Hessian 做 SPD projection，全靠惯性项 + 行列式阈值 + 单调下降三层兜底（3.6）。那是因为 strand 只有弹簧、只有一种会变负的特征值，惯性项几乎总能压住。可一旦上了 Newton 的 production 规模——cloth 的 Neo-Hookean membrane、tet 的 volumetric、bending、还有接触——元件的 Hessian 不定起来更凶、更频繁。这一节看 Newton 怎么在每个元件层面把这件事按死：把 per-element Hessian 直接投成 SPD，再配上一个干净的 Rayleigh damping。

为什么 production 要在元件层面投影

回忆 1.5 那个核心事实：一根被压缩的弹簧，Hessian 在垂直方向特征值 $k(1-l_0/l)$ 会变负，矩阵不定——这是 buckling（屈曲）方向的忠实编码。strand 里只有这一种元件、这一种负特征值来源，所以 TinyVBD 敢赌「惯性项 $\frac{m_i}{h^2}\mathbf I$ 多半能把它顶回正」。

但 Newton 的 cloth 用的是 stable Neo-Hookean membrane，能量里那个体积/面积项 $(J_s-\alpha)$ 在压缩（$J_s<\alpha$，面积被压小）时同样会让 Hessian 出现负特征值——和压缩弹簧一模一样的毛病，只是来自一个更复杂的能量。tet 的 volumetric、bending 也各有自己的不定方向。把这么多元件的负特征值叠到一个顶点的 $\mathbf H_i$ 上，惯性项不一定每次都兜得住。Newton 的选择不是赌，而是在每个元件装配进 $\mathbf H_i$ 之前，先把它那一块的 Hessian 投成 SPD（positive semi-definite）。

membrane 的 SPD-clamp：max(0, ·)

以 membrane 为例，Newton 的实现里那一步极其朴素。能量 $\psi=\frac{\mu}{2}(I_c-2)+\frac{\lambda}{2}(J_s-\alpha)^2$ 装配 per-vertex Hessian 时，体积项贡献一个标量系数 $s=\lambda_\text{NH}(J_s-\alpha)$；Newton 直接对它做一次下截断：

$$s_\text{clamp}=\max\!\big(0,\;\lambda_\text{NH}\,(J_s-\alpha)\big).$$

当面积被压缩到 $J_s<\alpha$，$s$ 本会变负、把 Hessian 块拉向不定；$\max(0,\cdot)$ 把它夹到 0，于是这一块退化成「只保留恒正定部分」的 PSD 近似。代码注释把它讲得很白：这是 membrane per-vertex 块的 tight PSD clamp。装配进 $\mathbf H_i$ 的，永远是已经投过影的块——$\mathbf H_i$ 从源头就难再不定。

particle_vbd_kernels.py
# --- Hessian (per-vertex 3x3, SPD-projected) ---
# max(0, s) is the tight PSD clamp for the membrane per-vertex block.
s_clamp = wp.max(0.0, s)   # s = lmbd_nh * (J_s - alpha)
r = s_clamp * inv_J_s
c1 = lmbd_nh - r

手算一遍钳取：压缩元件的两个特征值

$\max(0,\cdot)$ 听着抽象，代两个特征值进去就一目了然。复用 1.5 那个弹簧 Hessian $\mathbf H_{aa}=k\big[\mathbf I-\frac{l_0}{l}(\mathbf I-\frac{\mathbf d\mathbf d^\top}{l^2})\big]$——它有两类特征向量，特征值闭式可写：沿弹簧方向 $\hat{\mathbf d}$ 的是 $k$，垂直方向的是 $k(1-l_0/l)$。取一个被压缩的元件（当前长 $l<l_0$，比如 $l_0/l=1.25$），并把 $k$ 归一到 $1$ 让数字干净：

$$\lambda_\parallel=k=+1\ (\text{沿 }\hat{\mathbf d}),\qquad \lambda_\perp=k\Big(1-\frac{l_0}{l}\Big)=1\times(1-1.25)=-0.25\ (\text{垂直 }\hat{\mathbf d})$$

沿弹簧方向永远硬（$+1$，拉/压都抗）；垂直方向因为被压缩而翻负（$-0.25$）——这就是 buckling 方向，矩阵在这里不定。没有 SPD 投影，牛顿步就可能沿这个负特征值方向跑出上坡。Newton 的做法是把惹事的那一项系数钳到非负，等价于把这个负特征值拍到 0：

$$\lambda_\perp^{\text{clamp}}=\max(0,\,-0.25)=0\quad\Rightarrow\quad \text{特征值变成}\ \{+1,\ 0,\ 0\}\ (\text{半正定})$$

钳完这块 Hessian 只剩半正定（最小特征值 $0$，还差一口气到正定）。这时 VBD 自带的惯性项接力：$\frac{m}{h^2}\mathbf I$ 给每个方向都加一个严格为正的底。沿用全书参数 $m=1,\ h=\tfrac{1}{60}$，惯性底是 $\frac{m}{h^2}=3600$，于是合成后的特征值是

$$\{+1,0,0\}+3600\cdot\{1,1,1\}=\{3601,\ 3600,\ 3600\}\;>\;0$$

三个特征值全严格为正——$\mathbf H_i$ 正定，$3\times3$ 求逆稳稳可解、$\Delta\mathbf x$ 是可靠下降方向。两道一起看：SPD 投影先把 $-0.25$ 拍到 $0$（半正定），惯性项再把 $0$ 顶成 $3600$（正定）。对照 strand：TinyVBD 跳过第一道、只靠惯性项硬扛——单根弹簧时 $3600$ 远大于 $|{-0.25}|$（甚至 $1.4$ 那种 $k=100$ 的 $\lambda_\perp=-100$ 也压得住，因为 $3600\gg100$），所以赌得过；但 production 多元件把许多负特征值叠到一个顶点上时，惯性底未必次次够，才必须先投影。

这组特征值是自洽示例，量级随材料/单位变

$+1/-0.25$、惯性底 $3600$ 是为手算挑的干净自洽数（$k$ 归一到 $1$、用全书的 $m=1,h=\tfrac1{60}$）。真实 membrane 的负特征值来自 $s=\lambda_\text{NH}(J_s-\alpha)$、量级随 Lamé 参数与网格尺度缩放，可能远大于这里的 $0.25$。能写准也值得记的是机理：压缩 → 垂直特征值翻负 → $\max(0,\cdot)$ 拍到 0（半正定）→ 惯性项 $\frac{m}{h^2}\mathbf I$ 抬成正定。具体数字会变，这条链不变。

interactive拖动弹簧，看垂直特征值怎么翻负
把弹簧拖到压缩（当前长 < rest 长），看垂直方向的特征值由正转负——正是上面手算的 $-0.25$ 那个分量的现场版。这块不定的 Hessian，就是 Newton 要 $\max(0,\cdot)$ 投影、再靠惯性项 $\frac{m}{h^2}\mathbf I$ 顶回正定的对象。

stable Neo-Hookean：让 rest state 无应力

上面那个 $\lambda_\text{NH}$ 不是随手写的 Lamé 参数。Newton 按 Smith 2018 §3.4 的 stable Neo-Hookean 约定，把用户给的 Lamé 参数换算成 NH 能量里真正用的参数：

$$\mu_\text{NH}=\mu_\text{Lamé},\qquad \lambda_\text{NH}=\lambda_\text{Lamé}+\mu_\text{Lamé}.$$

再配上 $\alpha=1+\mu_\text{NH}/\lambda_\text{NH}$，这套约定保证一件要紧的事：在 rest configuration（$F$ 为等距、$J_s=1$）下能量的梯度恰好为零，也就是静止状态无应力（stress-free）。否则材料一放进场景就自己鼓一下、缩一下，rest 形状都站不稳。这是 production 弹性必须做对、而 strand 那种纯 Hookean 弹簧天然不需要操心的细节——弹簧的 $E=\frac12 k(l-l_0)^2$ 在 $l=l_0$ 处梯度本就为零。

naive（不投影）Hessian 会怎样拖垮 production 求解器

设想 Newton 不做这一步：直接把元件原始的、可能不定的 Hessian 累加进 $\mathbf H_i$。一旦合成的 $\mathbf H_i$ 带了负特征值，$\mathbf H_i^{-1}\mathbf f_i$ 就可能指向上坡——这一步不是在降 $G$，而是在升 $G$；轻则这个顶点抖、收敛变慢，重则在高刚度大网格上把整块 cloth 推爆。strand 上靠「惯性项 + det 阈值跳过」兜底，是因为不定来得稀、来得弱；放大到上百万顶点、多种元件，这种侥幸不再成立。SPD projection 的作用，就是从每个元件源头掐掉负特征值，让累加出来的 $\mathbf H_i$ 结构性地保持半正定，每一步都是可靠的下降方向。而 VBD 自带的惯性项 $\frac{m_i}{h^2}\mathbf I$ 仍在帮忙——它给 PSD 的 $\mathbf H_i$ 再抬一个严格为正的底，把「半正定」顶成「正定」，$3\times3$ 求逆稳稳可解。两道一起上，才是 production 该有的保险。

从 TinyVBD 看 同一个负特征值，两种处理哲学

你在 strand 上亲眼见过：压缩弹簧 → 垂直方向特征值 $k(1-l_0/l)$ 变负 → 局部 $\mathbf H$ 不定（1.5 的 spring-hessian 部件拖一下就能看到）。TinyVBD 的回应是「不投影、赌惯性项」（3.6）：靠 $\frac{m_i}{h^2}\mathbf I$ 抬底、det 太小就跳过、单调下降兜大局——三层兜底，省掉 SPD 投影的开销，在单根 strand 上够用。Newton 面对的是同样的负特征值（只是从 Neo-Hookean、tet、bending 多个能量里冒出来、更密更猛），但它不能赌：在每个元件上提前 $\max(0,\cdot)$ 投成 SPD，再让惯性项接力。一句话——$\min G(\mathbf x)$ 没变，压缩元件会让 Hessian 不定这件事也没变；变的是「单根赌惯性」升级成了「整网在 GPU 上逐元件 SPD 投影 + 惯性接力」。

Rayleigh damping：D = kd·ke

稳定性还有另一面：耗散。TinyVBD 对阻尼的处理很轻——弹簧带一个 damping 项，本质是把速度相关的耗散塞进力里，简单直接。Newton 走的是更规整的 Rayleigh damping 约定：阻尼系数不是一个独立的物理量，而是正比于刚度——

$$D=k_d\cdot k_e,$$

其中 $k_e$ 是元件刚度，$k_d$ 是一个无量纲的 Rayleigh 系数（solver_vbd.py 注明 VBD 把 kd 解释为 dimensionless Rayleigh coefficient）。这样写有两个好处：一是阻尼自动随刚度缩放，硬的地方阻尼也大、软的地方阻尼也小，不用为每种材料单独标定一个绝对阻尼；二是 $k_d$ 无量纲，跨场景搬运不会因单位变化而错。membrane 与 volumetric 各自有对应的 Frobenius/trace 形式耗散项接进这套约定。

无量纲约定的一个反作用力

正因为 $D=k_d\cdot k_e$ 把阻尼绑在刚度上，一个看起来不大的 $k_d$ 配上很硬的 $k_e$ 也可能耗散过猛。Newton 文档就专门提醒：某些 joint limit 的默认 kd（如 $10^1$）在这套 Rayleigh 约定下会产生 excessive damping，用到时要显式调小。这是「约定优雅」换来的代价——你得记住它是相对刚度的系数，不是绝对阻尼值。

这一节你要带走的

三句话：(1) production VBD 在每个元件把 Hessian 投成 SPD（membrane 用 $s_\text{clamp}=\max(0,\lambda_\text{NH}(J_s-\alpha))$ 这种 tight PSD clamp），从源头掐掉压缩带来的负特征值，再让 VBD 自带的惯性项 $\frac{m_i}{h^2}\mathbf I$ 把半正定顶成正定；(2) stable Neo-Hookean 用 $\lambda_\text{NH}=\lambda_\text{Lamé}+\mu_\text{Lamé}$（Smith 2018 §3.4）保证 rest state 无应力；(3) 阻尼用无量纲 Rayleigh 约定 $D=k_d\cdot k_e$，比 TinyVBD 把阻尼直接塞进弹簧力更规整、可移植，但要记得它是相对刚度的系数。

到这里，元件的力、Hessian、SPD 投影、阻尼都齐了——一个 cloth/soft body 顶点该装配的能量项全部就位。但 production 仿真还差最后、也最难的一块：物体不能互相穿插。下一节 10.3 进入 self-collision 与 penetration-free——Newton 怎么用 BVH 检测 vertex-triangle / edge-edge 接触、怎么在每色 sweep 后做保守截断保证不穿透，而这正是 TinyVBD 那根 strand 完全没有的一整块世界。
