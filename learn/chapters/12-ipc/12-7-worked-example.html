手把手算一遍：一个接触对从距离到 Δx

前面把零件都讲齐了——距离（12-3）、barrier 求导与 PSD（12-4）、CCD line search（12-5）、摩擦（12-6）。这一节把它们在一个具体接触对上拼起来，用真实数字算一遍。我们仿照 4.3 解剖 solve() 的节奏：摆出一组数 → 每一步对一个公式 → 把中间值写出来让你能拿计算器复算 → callout 解释「为什么这样」。算完你会得到的不是一个结论，而是一种把握：从「两个几何元件隔着 0.30」到「这一步该把点往哪挪」，每一环都接得上、符号和方向都对。

摆出这一对：一个 2D point-segment

为了能手算，我们用一个 2D point-segment 作例子——它是 point-triangle 在二维里的类比，几何完全同构，但所有数都能写在一行里。设：

$$\mathbf p = (0,\ 0.30),\qquad \mathbf a = (-1,0),\quad \mathbf b = (1,0),\qquad \hat d = 0.40,\quad \kappa = 1$$

一个点 $\mathbf p$ 悬在线段 $ab$ 正上方 $0.30$ 处；barrier 的激活阈值 $\hat d=0.40$，刚度系数 $\kappa=1$（取 1 是为了让数字干净，真实场景 $\kappa$ 会大得多）。因为点离线段 $0.30 < \hat d = 0.40$，已经进入 barrier 的激活窗口，接触项要发力了。下面顺着 12-3 → 12-4 的次序，一步步算。

这组数是为讲解构造的自洽示例

声明在前：这一节的具体数字是为教学专门构造的自洽示例，非论文复现。barrier 解析式用 12-2 标注过的常见形式 $b=-(d-\hat d)^2\ln(d/\hat d)$（精确定义、单位约定以 Li et al. 2020 论文 PDF 为准）。我们刻意把能量这种「绝对量级会随 $\kappa$ 和单位变」的数标成示意，只对能写准的东西——距离、最近点、法向、权重、力的方向与符号——较真。重点不是某个小数，而是每一步怎么由上一步推出来。

第一步 · 距离与最近点

先回答 12-3 的几何问题：$\mathbf p$ 到线段的最近点 $\mathbf q$ 在哪？$\mathbf p$ 在线段正上方，竖直投下来正好落在原点，且原点在线段内部（$x=0\in[-1,1]$），所以

$$\mathbf q = (0,0),\qquad d = \lVert \mathbf p - \mathbf q\rVert = \lVert(0,0.30)\rVert = 0.30,\qquad \hat{\mathbf n} = \frac{\mathbf p-\mathbf q}{d} = \frac{(0,0.30)}{0.30} = (0,1)$$

最近点落在线段内部，写成 barycentric 组合 $\mathbf q=\lambda_a\mathbf a+\lambda_b\mathbf b$：原点是 $\mathbf a=(-1,0)$ 和 $\mathbf b=(1,0)$ 的正中点，所以

$$\lambda_a = \lambda_b = 0.5,\qquad \lambda_a+\lambda_b = 1\ \checkmark$$

四个量备齐：距离 $d=0.30$、最近点 $\mathbf q=(0,0)$、权重 $0.5/0.5$、单位法向 $\hat{\mathbf n}=(0,1)$（竖直向上，从线段指向点）。这就是后面所有步骤的输入。

第二步 · barrier 能量：代入看一眼量级

把 $d=0.30$、$\hat d=0.40$ 代进 12-2 的 barrier。逐个算中间值，每个都能在计算器上验：

$$\frac{d}{\hat d} = \frac{0.30}{0.40} = 0.75,\qquad \ln 0.75 \approx -0.288,\qquad (d-\hat d)^2 = (-0.10)^2 = 0.01$$
$$b = -(d-\hat d)^2\,\ln\!\Big(\frac{d}{\hat d}\Big) \approx -\,(0.01)\times(-0.288) \approx 2.9\times10^{-3}$$

读出来：$d<\hat d$ 时 $\ln(d/\hat d)<0$，前面的负号把它翻成正能量——barrier 在「储能」，像一根被压住的弹簧。这个数（约 $2.9\times10^{-3}$，再乘上 $\kappa=1$）现在还小，因为点离墙还有 $0.30$；一旦 $d\to0$，$\ln(d/\hat d)\to-\infty$，$b\to+\infty$，能量冲向无穷——这正是 12-2 拆过的「$\ln$ 端发散、$(d-\hat d)^2$ 端平滑接零」。

能量的绝对值别太当真，趋势才是真的

$2.9\times10^{-3}$ 这个数本身意义不大——它会随 $\kappa$ 和长度单位整体缩放。能写准也值得记的是：它是正的（在储能）、它随 $d$ 减小而单调暴涨。解析式仍以论文为准；我们要的是「代进去看它怎么随距离变」，不是把这个小数当物理常数。

第三步 · 力：方向沿 +n̂ 把点顶开

力是能量的负梯度。按 12-4 的链式法则，barrier 对 $\mathbf p$ 的力是

$$\mathbf f_{\mathbf p} = -\frac{\partial b}{\partial \mathbf p} = -\,b'(d)\,\frac{\partial d}{\partial \mathbf p} = -\,b'(d)\,\hat{\mathbf n}$$

$b'(d)$ 在 $d<\hat d$ 时为负（能量随 $d$ 减小而升，故对 $d$ 的导数为负），于是 $-b'(d)>0$，力沿 $+\hat{\mathbf n}=(0,1)$——竖直向上，把点 $\mathbf p$ 顶离线段。方向算得明明白白：朝着「让距离变大」的方向推。再看大小的趋势：$d$ 越小，$|b'(d)|$ 越大（barrier 越靠近 0 越陡），这股顶开的力就越凶——这就是为什么穿不过去。这里 $d=0.30$ 离 $\hat d=0.40$ 不远，曲线还平缓，所以力还温柔；真把点压到 $d=0.05$，同一个公式会吐出一股极大的排斥力。

第四步 · 落到顶点：点拿满，两端各分一半

这一对接触不只推 $\mathbf p$，线段的两个端点 $\mathbf a,\mathbf b$ 也要受力。按 12-3 的 barycentric 落点规则，把同一股标量力 $-b'(d)$ 按权重分给三个顶点：

$$\mathbf f_{\mathbf p} = -b'(d)\,\hat{\mathbf n},\qquad \mathbf f_{\mathbf a} = +b'(d)\,\lambda_a\,\hat{\mathbf n},\qquad \mathbf f_{\mathbf b} = +b'(d)\,\lambda_b\,\hat{\mathbf n}$$

代入 $\lambda_a=\lambda_b=0.5$、$\hat{\mathbf n}=(0,1)$，并记顶开点的那股力大小为 $F=-b'(d)>0$：

$$\mathbf f_{\mathbf p} = F\,(0,1),\qquad \mathbf f_{\mathbf a} = \mathbf f_{\mathbf b} = -\tfrac{1}{2}F\,(0,1)$$

读出物理：点 $\mathbf p$ 受 $+\hat{\mathbf n}$ 向上被顶开；线段两端各受 $-0.5\,\hat{\mathbf n}$向下的反作用力，权重正好是它们的 barycentric 系数 $0.5$。三股力相加 $F-0.5F-0.5F=0$——牛顿第三定律自动成立，接触对内部不产生净外力。因为这里 $\mathbf q$ 恰在中点，两端平分；要是把 $\mathbf p$ 移到偏 $\mathbf a$ 一侧，$\lambda_a$ 变大，$\mathbf a$ 就会分到更多——「谁离最近点近谁分得多」。

和 4.3 弹簧那个 ± 号是同一回事

回想 4.3 里弹簧力按端点取 ± 号、两端等大反向。这里一模一样，只是「两端平分」升级成「按 barycentric 权重分到多个顶点」。弹簧是 $\{+1,-1\}$ 两端，接触是 $\{+1,-\lambda_a,-\lambda_b\}$ 多端——权重之和让净力归零的逻辑没变。你在 4.3 建立的肌肉记忆，在这里直接复用。

第五步 · Hessian → PSD 投影 → 合成 → 这一步的 Δx

力定了方向，还差曲率才能解牛顿步。barrier 的 Hessian 由 12-4 的两项构成：一项来自 $b''(d)\,\hat{\mathbf n}\hat{\mathbf n}^\top$（沿法向的刚度，半正定），一项来自 $b'(d)\,\partial^2 d/\partial\mathbf x^2$（距离的二阶项，不保证半正定）。后者可能带来负特征值，让这块 Hessian 不定——直接拿去解牛顿步会跑出上坡方向。

所以照搬你在 10.2 学过的招：PSD projection，把这块接触 Hessian 的负特征值钳到零（或一个小正数），保证它半正定、保证解出来是下降方向。钳完，这块接触 Hessian 和惯性项 $\frac{m}{h^2}\mathbf I$、弹性项一起累加进同一个 $\mathbf H$，力一起进同一个 $\mathbf f$，解一步：

$$\Delta\mathbf x = \mathbf H^{-1}\mathbf f$$

对 $\mathbf p$ 这个顶点而言，$\mathbf f$ 里接触贡献的那一份是 $+\hat{\mathbf n}=(0,1)$ 方向，惯性项可能在把它往别处拽，但接触这一项明确地朝「抬离线段」的方向出力。合成后这一步的 $\Delta\mathbf x_{\mathbf p}$ 因此带一个向上的分量——把点抬离线段、把距离 $d$ 往 $0.30$ 之上推。具体抬多少是定量的（取决于 $\kappa$、质量、其余各项怎么拔河），我们不编精确小数；但方向是确定的、符号是对的：barrier 进来就是为了把 $d$ 顶大，这一步 $\Delta\mathbf x$ 就让它顶大。

再补一句把全链接上：解出 $\Delta\mathbf x$ 后并不立刻全盘接受——还要过 12-5 的 CCD line search，求一个安全步长 $\alpha\in(0,1]$，保证沿 $\Delta\mathbf x$ 走 $\alpha$ 步后仍没穿过线段。我们这个点本来就在外侧、$\Delta\mathbf x$ 又是抬离方向，CCD 多半放行整步（$\alpha=1$）；但机制始终在那兜底，这正是 IPC 「intersection-free 是保证而非兜底」的来源。

回头看这条链

把五步连起来念一遍，就是 IPC 一个接触对的完整生命：几何给出 $d=0.30$、$\hat{\mathbf n}=(0,1)$、权重 $0.5/0.5$（12-3）→ 代进 barrier 得正能量 $\approx2.9\times10^{-3}$、确认在激活窗口内（12-2）→ 链式法则得力 $-b'(d)\hat{\mathbf n}$，方向沿 $+\hat{\mathbf n}$ 顶开点（12-4）→ 按 barycentric 把力分成点拿满、两端各 $-0.5\hat{\mathbf n}$，净力为零（12-3）→ Hessian 做 PSD 投影、与惯性/弹性合成、解 $\mathbf H^{-1}\mathbf f$，得到一步把点抬离线段的 $\Delta\mathbf x$，再经 CCD line search 截到可行域内（12-4/12-5）。每一环的输入都是上一环的输出，符号和方向全程自洽。这就是把前面分散的零件，在一对真实接触上拼成整机。

interactive把点设到 (0, 0.30)，对照这一节的算法
把点 $\mathbf p$ 拖到线段正上方（接近 $(0,0.30)$ 的位置），你会看到和上面计算同一张图：最近点 $\mathbf q$ 落在中点、$d\approx0.30$、$\hat{\mathbf n}$ 竖直向上、两端梯度箭头等长（权重各 $0.5$）。横向挪点让 $\mathbf q$ 偏向一端，看权重怎么失衡、哪端分到更长的箭头——正是第四步「谁近谁分得多」的现场版。

这一节你要带走的

三件事：(1) 一个接触对从距离到 $\Delta\mathbf x$ 是一条每环都接得上的链——几何（$d,\hat{\mathbf n}$，barycentric）→ barrier 能量 → 链式法则求力 → 按权重落到顶点 → Hessian 经 PSD 投影后与惯性/弹性合成解牛顿步 → CCD line search 截到可行域；本节用 $\mathbf p=(0,0.30)$、$d=0.30$、$\hat{\mathbf n}=(0,1)$、$\lambda=0.5/0.5$ 把每环的数都摆了出来。(2) 能写准的写准（距离、法向、权重、力的方向与符号、净力为零），会随单位/参数缩放的标成示意（能量绝对值、$\Delta\mathbf x$ 精确大小）——这是诚实，不是含糊。(3) 这组数是为讲解构造的自洽示例（非论文复现），barrier 形式以论文为准；重点是方向和连接，不是某个小数。
