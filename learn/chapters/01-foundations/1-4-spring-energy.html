弹簧的能量、力与 Hessian

前面 $E(\mathbf x)$ 一直是个抽象符号。这一节把它落到一根真实的 Hookean 弹簧上：写出能量、求出力、再求出 Hessian——然后撞见一个关键事实：弹簧被压缩时，Hessian 在垂直方向的特征值会变负、矩阵变得不定。这正是 solve() 里 h_1_1 那一行，也是 VBD 局部求解必须直面的硬骨头。

一根弹簧的能量 E(x)

取一根弹簧，两端点 $\mathbf x_a,\mathbf x_b$，rest length 为 $l_0$，stiffness 为 $k$。记方向向量与当前长度

$$\mathbf d=\mathbf x_a-\mathbf x_b,\qquad l=\lVert\mathbf d\rVert.$$

Hookean 弹簧的能量就是「拉伸量的平方」乘以半刚度：

$$E=\tfrac12 k\,(l-l_0)^2.$$

它在 $l=l_0$（恰好等于 rest 长度）时为零、为最小，往两边——无论拉长还是压短——都涨上去。把整根 strand 上所有弹簧的 $E$ 加起来，就是 1.2 那个 $G(\mathbf x)$ 里的弹性势能项。下面这根弹簧的全部物理，都从这一行长出来。

力 = 能量的负梯度

力是能量对位置的负梯度 $\mathbf f=-\partial E/\partial\mathbf x_a$。链式法则：$\partial E/\partial l=k(l-l_0)$，而 $\partial l/\partial\mathbf x_a=\mathbf d/l$（单位方向）。合起来，作用在端点 $a$ 上的力是

$$\mathbf f_a=-\,\frac{\partial E}{\partial\mathbf x_a}=k\,\frac{l_0-l}{l}\,\mathbf d,$$

端点 $b$ 取负号（作用力与反作用力）。读一下符号：当 $l>l_0$（被拉长），$l_0-l<0$，力沿 $-\mathbf d$，把 $a$ 往 $b$ 拉回去；当 $l<l_0$（被压短），$l_0-l>0$，力沿 $+\mathbf d$，把 $a$ 往外推开。弹簧总想回到 rest 长度，符合直觉。这一行正是 solve() 里 ±stiffness*(l0-l)/l*diff 的来历。

Hessian：力对位置的导数

VBD 每个顶点要做一步牛顿，光有力 $\mathbf f$ 不够，还要力的导数——也就是能量的二阶导 Hessian。对端点 $a$ 自身的 $3\times3$ 块（在二维里就是 $2\times2$，道理一样），把上面的力再对 $\mathbf x_a$ 求一次导，得到

$$\mathbf H_{aa}=k\Big[\mathbf I-\frac{l_0}{l}\Big(\mathbf I-\frac{\mathbf d\,\mathbf d^\top}{l^2}\Big)\Big].$$

这个式子初看吓人，其实结构很清楚。$\frac{\mathbf d\mathbf d^\top}{l^2}$ 是「沿弹簧方向」的投影矩阵，$\mathbf I-\frac{\mathbf d\mathbf d^\top}{l^2}$ 则是「垂直于弹簧方向」的投影。换句话说，$\mathbf H_{aa}$ 在沿弹簧和垂直弹簧两个方向上各有一个特征值——它们的故事完全不同。

两个特征值：一个永远正，一个会变负

把 $\mathbf H_{aa}$ 分别作用在「沿 $\mathbf d$」和「垂直 $\mathbf d$」的方向上，特征值一眼就读出来：

  • 沿弹簧方向：垂直投影那项为零，剩 $\mathbf H_{aa}\mathbf d=k\,\mathbf d$，特征值 $=k$。永远是正的，且就等于刚度——沿着弹簧拉伸，永远是「越拉越费力」的稳定方向。
  • 垂直弹簧方向：沿弹簧投影那项为零，特征值 $=k\big(1-\tfrac{l_0}{l}\big)$。它的符号取决于 $l$ 和 $l_0$ 的大小：拉伸（$l>l_0$）时为正；恰好 rest（$l=l_0$）时为零；压缩（$l<l_0$）时 $\frac{l_0}{l}>1$，于是这个特征值变负。
$$\lambda_{\parallel}=k\;(>0\ \text{恒成立}),\qquad \lambda_{\perp}=k\Big(1-\frac{l_0}{l}\Big)\;\xrightarrow{\,l<l_0\,}\;\text{负}.$$

一个特征值正、一个负，意味着压缩状态下 $\mathbf H_{aa}$ 不再正定，而是indefinite（不定）。物理上这正是buckling（屈曲）方向：一根被压短的杆，往侧向偏一点点能量反而下降，所以它不愿待在直的状态、想弯掉。负特征值忠实地编码了这种「侧向不稳定」。

代码对照：这就是 solve() 里的 h_1_1

上面那个 $\mathbf H_{aa}$ 不是黑板演算，它一字不差地躺在 TinyVBD 的 solve() 里：

main.cpp
Vec3 diff = strand.mVertPos.col(v1) - strand.mVertPos.col(v2);   // d = x_a - x_b
FloatingType l  = diff.norm();                                   // l = |d|
FloatingType l0 = strand.orgLengths(edgeId1);                    // rest length l0
Mat3 h_1_1;
h_1_1 = stiffness_ * (Mat3::Identity()
        - (l0 / l) * (Mat3::Identity() - diff * diff.transpose() / (l * l)));
h += h_1_1;

逐项对得上：diff 是 $\mathbf d$，l 是 $l$，l0 是 $l_0$，diff*diff.transpose()/(l*l) 就是 $\frac{\mathbf d\mathbf d^\top}{l^2}$。整句 h_1_1 = stiffness*(I - (l0/l)*(I - dd^T/l^2)) 与公式 $\mathbf H_{aa}=k[\mathbf I-\frac{l_0}{l}(\mathbf I-\frac{\mathbf d\mathbf d^\top}{l^2})]$ 完全一致。4.3 会把 solve() 整段逐行读透，这里你只需认出：那一行真的可能给你一个不定矩阵。

从 GAMES103 看 这就是隐式积分要做「SPD 投影」的原因

GAMES103 讲隐式积分常会提一句：组装出的 Hessian（或 $M-h^2\nabla\mathbf f$）有时不正定，Newton 一步可能走错方向甚至发散，于是要做 SPD projection——把负特征值强行拍正，或退回 line search。负特征值从哪来？正是上面这个压缩弹簧的 $\lambda_\perp<0$。你当年「为保险起见投一下影」的那个操作，针对的就是这一节亲手算出的不定性。

惯性项 m/h²·I 是怎么帮上忙的

好消息是：VBD 每个顶点真正要解的，不是孤零零的弹簧 Hessian，而是它加上惯性项之后的局部 Hessian（见 1.3）：

$$\mathbf H_i=\underbrace{\frac{m_i}{h^2}\mathbf I}_{\text{恒正定}}+\sum_{j}\mathbf H_{aa}^{(j)}.$$

惯性项 $\frac{m_i}{h^2}\mathbf I$ 是各向同性的正定矩阵，它给每个特征值都加上一个正的 $\frac{m_i}{h^2}$ 偏置。于是那个会变负的 $\lambda_\perp=k(1-l_0/l)$，实际进入求解的是 $\frac{m_i}{h^2}+k(1-l_0/l)$——只要质量不太小、时间步不太大，这个偏置足以把它顶回正数，整个 $\mathbf H_i$ 重新正定，牛顿步方向可靠。这就是 1.2 那句「惯性项保证问题不病态」的具体兑现。

但偏置不是万能的

当弹簧极硬（$k=10^8$）而质量很轻、或时间步偏大时，$\frac{m_i}{h^2}$ 这点正偏置可能压不住 $k(1-l_0/l)$ 的负，局部 $\mathbf H_i$ 仍可能不定。Newton 类方法到这里要么投影、要么 line search 才能保稳。VBD 选了另一条路：它不强制把局部 Hessian 投成 SPD，而是靠一套别的机制（行列式阈值、单调下降保证）照样稳。这条路怎么走，正是 3.5 的主题。

这一节你要带走的

三句话：(1) 一根 Hookean 弹簧 $E=\tfrac12 k(l-l_0)^2$，力 $\mathbf f=k\frac{l_0-l}{l}\mathbf d$，Hessian $\mathbf H_{aa}=k[\mathbf I-\frac{l_0}{l}(\mathbf I-\frac{\mathbf d\mathbf d^\top}{l^2})]$，就是 solve() 的 h_1_1；(2) 它沿弹簧的特征值恒为 $k$，垂直方向为 $k(1-l_0/l)$——压缩时变负，$\mathbf H$ 不定（屈曲方向）；(3) 永远正定的惯性项 $\frac{m}{h^2}\mathbf I$ 给所有特征值加正偏置，多数情况下把它顶回正定，但不是万能。

带着「局部 Hessian 可能不定」这个事实，我们就有了进入 Part 3 的票。3.5 会讲 VBD 如何不靠 SPD 投影也能稳，4.3 会把 solve() 里这套力与 Hessian 的装配逐行读完。Part 1 的地基到此打齐——隐式欧拉是什么、为什么、长什么样，你都拿到手了。
