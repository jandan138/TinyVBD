读懂 TinyVBD

从你熟悉的「粗暴调位置」一路走到 Vertex Block Descent
一本娓娓道来的交互式教程 · 基于 SIGGRAPH 2024 论文 Vertex Block Descent 与其极简实现 TinyVBD

这本书是写给谁的？

你上过 GAMES103，写过弹簧质点；你对 PBD 的理解停留在「把粒子位置硬掰到约束上、反复迭代」；你对隐式欧拉背后的数学本质还有点发虚。这本书就从这里出发。

我们会把 TinyVBD 这份 ~400 行的 strand 仿真代码逐行读透，并在路上补齐 implicit Euler 的变分形式、PBD / XPBD / Projective Dynamics 的谱系，最终理解 VBD 为什么又快又稳。每一章都配可交互的动画与实验。

怎么开始？

从 0-1 这本书要带你去哪 开始顺着读；或按 / 搜索任意小节。顶栏 ☾ 切换深色模式。
